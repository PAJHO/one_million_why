## 关于本书

本项目致力于构建一本开源免费、新手友好的数据结构与算法入门书。

- 全书采用动画图解，内容清晰易懂、学习曲线平滑，引导初学者探索数据结构与算法的“知识地图”；
- 源代码可一键运行，帮助读者在实践练习中提升编程技能，了解算法工作原理和数据结构底层实现；
- 鼓励读者在章节讨论区互帮互助、共同进步，提问与评论一般能在两日内得到回复；

如果本书对您有所帮助，请点个 Star :star: 支持一下，谢谢！
